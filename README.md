# Contact Saver UI
A sleek and responsive Contact Management application built with React and Tailwind CSS. 
This project allows users to save contacts with names, numbers, and profile images in a beautiful card layout, and also user can delete their Contacts.

Features
1. Add Contacts: Easily add contact details including a profile picture URL.
2. Dynamic Rendering: Contacts are displayed in real-time as soon as they are added.
3. Fully Responsive: Works perfectly on mobile, tablets, and desktops using Tailwind's responsive utilities.
4. Delete Contacts: Easily delete contact with the help of delete icon.

Tech Stack
1. Frontend: React.js
2. Styling: Tailwind CSS
3. Icons: Lucide-react

What I Learned
1. Handling Forms and State in React.
2. Working with Arrays and the Map function to render dynamic contacts.
3. Creating Responsive Layouts using Tailwind CSS breakpoints.
4. Two-Way Binding: control form inputs with input value and onChange() method.
